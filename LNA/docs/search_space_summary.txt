4. Active Circuit Constraints
-----------------------------

Global removals before role-specific filtering:

- remove every L-L pair
- remove every C-C pair
- networks after these global removals:
  108 - 6 series L-L - 6 parallel L-L - 10 series C-C - 10 parallel C-C = 76

Gate network:

- must come from the passive database
- must contain exactly one inductor
- allowed topology: single L
- a fixed input coupling capacitor is inserted before the gate inductor

Source network:

- may come from the passive database or be None
- None means short
- single C is not allowed
- series R-L is not allowed
- parallel R-L is not allowed
- parallel R-C is not allowed
- any series network containing a capacitor is not allowed
- any C-C network is not allowed
- any L-L network is not allowed
- allowed topologies: R, L, parallel L-C, short

Load network:

- must come from the passive database
- single C is not allowed
- series R-L is not allowed
- parallel R-L is not allowed
- parallel R-C is not allowed
- any series network containing a capacitor is not allowed
- any C-C network is not allowed
- any L-L network is not allowed
- allowed topologies: R, L, parallel L-C

Feedback network:

- fixed to None
- None means open
- no feedback network is inserted

Fixed circuit elements still present:

- gate bias resistor remains fixed
- output load resistor remains fixed
- input coupling capacitor is a fixed MIM capacitor:
  sky130_fd_pr__cap_mim_m3_2 w=10 l=10 mf=1
  This fixed input capacitor is not part of the optimization database.
- output coupling capacitor is a fixed MIM capacitor:
  sky130_fd_pr__cap_mim_m3_2 w=10 l=10 mf=1
  This fixed output capacitor is not part of the optimization database.

7. Current Make Commands
------------------------

Generate the current passive-network database:

  make generate-passive-network-multiobjective

Generate a limited circuit set:

  make generate-circuits MAX_CIRCUITS=10

Generate and simulate a limited circuit set:

  make generate-and-simulate-circuits MAX_CIRCUITS=10 SIM_MAX_CIRCUITS=10

Simulate already generated circuits:

  make simulate-circuits SIM_MAX_CIRCUITS=10

Analyze existing measures without rerunning ngspice:

  make analyze-circuits SIM_MAX_CIRCUITS=10
